Installation
============

Dependencies
------------

sklearn-genetic requires:

- Python (>= 3.6)
- scikit-learn (>= 0.23)
- deap (>= 1.0.2)
- numpy
- multiprocess

User installation
-----------------

The easiest way to install sklearn-genetic is using :code:`pip`

.. code:: bash

    pip install sklearn-genetic

or :code:`conda`

.. code:: bash

    conda install -c conda-forge sklearn-genetic
